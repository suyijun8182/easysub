## [1.7.0]

### Added
- **通知渠道扩到 14 个**：新增 Server酱 / 企业微信 / 钉钉 / Discord / Slack / ntfy / Gotify。
- **月度预算**：设置里设预算，仪表盘显示进度条与超支预警。
- **免打扰时段**：非紧急提醒在时段内暂缓（到期当天/次日仍发）。
- **每周汇总推送**：按设定的星期几汇总即将续费/已过期项。
- **试用期 / 取消截止日 / 付款卡到期** 字段与到点提醒，避免被自动扣费。
- **CSV 批量导入 / 导出** 订阅。

### Fixed
- 无

---

## [1.6.0]

### Added
- **本地定时自动备份**：每天凌晨自动把整站数据导出到 `data/backups/`（保留最近 14 份），
  管理员可在设置页查看列表并「立即备份」。多一层防丢数据保险。
- **日历订阅 (.ics)**：设置页生成私有订阅链接，把续费日订阅到 Apple / Google / Outlook
  日历，续费日自动进日历并原生提醒（可重置链接）。
- **跟随系统深色模式**：主题新增「Auto」，随系统浅色/深色自动切换。
- **订阅列表搜索**：订阅管理页可按名称 / 套餐 / 备注 / 官网快速筛选。

## [1.5.2]

### Fixed
- **修复点击 Webhook 标签导致「通知」配置整块消失**：根因是 Webhook 模板提示文案里的
  `{{ }}` 占位符触发 vue-i18n 消息编译报错。改写文案，占位符示例改由输入框展示。

### Changed
- 「按分类浏览」服务库**继续大幅扩充**：在运营商之外，为 流媒体 / 音乐 / AI / 游戏 /
  VPS / 云存储 / 软件 / 域名 / 教育 / 新闻 / 健身 / 会员 各分类新增约 120 项常用服务。
- Docker Hub 说明与 README 同步为多渠道通知描述。

## [1.5.1]

### Changed
- 通知渠道配置从独立页面**收回到「设置」页**内（作为一个区块，含 7 个渠道标签页）。
- 添加订阅「提前提醒」默认改为**倒数 7 天每天提醒**（`7,6,5,4,3,2,1`），仍可自定义。
- 「按分类浏览」服务库**大幅扩充运营商 / eSIM**（英美欧日韩港台印澳加等约 55 家）。

### Fixed
- 修复通知配置 **Webhook 标签页打开空白** 的问题（配置字段缺失时的渲染守卫）。
- 修复 CONTRIBUTING.md 中 `你的用户名` 占位坏链；补充缺失的 CODE_OF_CONDUCT.md。

## [1.5.0]

### Added
- **多通知渠道**：在 Telegram 之外新增 飞书 / QQ / Bark / Email / Pushplus / Webhook，
  独立「通知配置」页（7 标签页），每渠道可单独启用与测试（闭合 issue #1 Bark 支持）。
- 备份/恢复（单用户 + 管理员整站）纳入通知渠道配置，重新部署后不丢失。

### Changed
- 仪表盘「分类总览」每个分类内订阅改为按到期时间升序排列。
- Telegram 配置从「设置」页迁移到独立「通知配置」页（设置页改为跳转入口）。

### Fixed
- **数据库断线自愈**：宿主机/容器异常重启后，app 若早于 MySQL 启动不再永久掉入
  安装向导；启动多次重试 + 请求到达时自动重连，前端自愈期间不再误弹向导。

## [1.0.0] - 2026-06-19

### Added

#### 核心功能
- ✅ 多用户系统 - JWT 鉴权，管理员和普通用户
- ✅ 订阅管理 - 支持创建、编辑、删除订阅
- ✅ 周期设置 - 支持按日/周/月/年计费，支持一次性买断
- ✅ 自动续费提醒 - APScheduler 定时任务 + Telegram 推送
- ✅ 仪表盘 - 总览支出、即将到期项目、最近订阅

#### 多语言和本地化
- 支持中文 / English / Русский
- vue-i18n 国际化框架
- 用户可选择语言

#### 多主题支持
- 5 个预设主题：浅色、深色、海洋、森林、紫罗兰
- CSS 变量实现主题切换

#### 货币管理
- 支持全球主流货币（USD、CNY、EUR 等）
- 实时汇率更新（open.er-api.com）
- 用户自定义货币功能
- 日期缓存，避免频繁 API 调用

#### 分类管理
- 预设分类：流媒体、AI、游戏、VPS、电信运营商等
- 用户可自定义分类
- 分类排序功能

#### 付款方式
- 预设付款方式：信用卡、Apple Pay、支付宝、微信、PayPal 等
- 用户可自定义付款方式

#### 日历视图
- 苹果风格日历
- 显示所有续费日期
- 交互式日期选择

#### 报表分析
- 支出洞察 - 总支出、平均支出、支出趋势
- 分类排行 - 按分类统计支出
- 永久购买统计
- 即将续费提醒
- 已过期订阅查看
- 按月/年展示

#### 自定义图标
- Emoji 选择
- 图片上传功能
- 本地图标库

#### Telegram 通知
- 创建 Bot Token 验证
- Chat ID 自动获取
- 自定义提醒天数
- 测试消息发送
- 通知日志记录

#### 数据库
- MySQL 8 支持
- 网页安装向导配置连接
- 自动建表和初始化
- 数据库迁移支持

#### 部署
- Docker Compose 一键部署
- Dockerfile 多阶段构建
- Caddy 反向代理 + 自动 HTTPS
- NAS 部署支持（TrueNAS、fnOS、群晖等）
- 环境变量配置

### Changed
- 项目名称：「手机保号通知」→ 「省心订阅 EasySub」
- 数据库架构完全重设计，支持外部 MySQL

## 版本发布规则

- **主版本号**（Major）：不兼容的 API 更改
- **次版本号**（Minor）：新增功能，向后兼容
- **修订版本号**（Patch）：Bug 修复，向后兼容
